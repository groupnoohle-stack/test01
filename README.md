# Test01 项目

## 简介

这是一个测试项目，包含简易在线计算器和 GitHub 集成功能测试报告。

## 项目内容

### 1. 简易在线计算器 (calculator.html)

一个功能完整的在线计算器，支持基本的数学运算。

**功能特性**：
- ✅ 基本运算：加法、减法、乘法、除法
- ✅ 小数点支持
- ✅ 连续计算功能
- ✅ 清除功能
- ✅ 键盘输入支持
- ✅ 响应式设计
- ✅ 美观的渐变背景和现代化 UI

**使用方法**：
1. 直接在浏览器中打开 `calculator.html` 文件
2. 使用鼠标点击按钮进行计算
3. 或使用键盘输入：
   - 数字键：0-9
   - 运算符：+、-、*、/
   - 等号：Enter 或 =
   - 清除：Esc 或 C
   - 退格：Backspace
   - 小数点：.

**技术栈**：
- HTML5
- CSS3（渐变、网格布局、动画效果）
- 原生 JavaScript（无依赖）
